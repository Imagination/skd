Welcome to skd

skd is a tool for the simple distribution of SSH-keys in a growing environment.

It is based on Django and uses the Paramiko SSH-Stack and the HTML5boilerplate template.

With skd you can create hosts, group them and link that group a group of users with keys to easily distribute all affected keys. skd generates a DSS-Keypair, that is saved to its database and used as authentication source for skd itself.

Now you can simply add a user with its key and press one button to grant the user access to all needed hosts.

See the README.md for details.
